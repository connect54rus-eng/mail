СПРАВКА ПО РЕДАКТОРУ

  • • Ж, К, Ч — кнопки форматирования текста (Жирный, Курсив, Подчеркнутый).
  • • URL — вставка гиперссылки в текст.
  • • ↶ — отмена последнего действия (аналог Ctrl+Z).
  • • Цифры (1-7) — размер шрифта.
  • • Цветные кружки — выбор цвета текста.
  • • Код ▼ — разворачивает окно с готовым кодом.
  • • Копировать — сохраняет результат кода в буфер.
  • • X — полная очистка редактора.
